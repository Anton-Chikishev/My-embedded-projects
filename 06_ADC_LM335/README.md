# Измерение температуры (Atmega32 + LM335)

## Задача
Измерение напряжения на датчике температуры LM335, преобразование АЦП в температуру и вывод на LCD.

## Алгоритм
1. Инициализация АЦП: включение, делитель частоты 1/64 (8 МГц / 64 = 125 кГц), опорное напряжение AVCC (5 В)
2. Выбор канала ADC3 (PA3)
3. Запуск преобразования, ожидание флага ADSC
4. Чтение регистра ADC, пересчёт в напряжение: `volt = cod * 5 / 1024` ≈ `cod * 0.00489`
5. Пересчёт напряжения LM335 в температуру (10 мВ/К): `temp = (volt - 2.73) / 0.01` (2.73 = 273 К)
6. Вывод кода АЦП и температуры на LCD

## Выводы МК
- PA3 — аналоговый вход от датчика LM335
- LCD на стандартных выводах (библиотека lcd1602wire4.h)

## Среда разработки
Microchip Studio, частота МК 8 МГц, библиотека lcd1602wire4.h
