# Переключение режимов свечения (Atmega32)

## Задача
По нажатию кнопки на PB0 переключается режим свечения светодиодов на порту D. Режимы:
0. Сдвиг одного огонька сверху вниз
1. Сдвиг двух огоньков навстречу друг другу
2. Зажигание всех светодиодов сверху вниз и гашение снизу вверх
3. Расхождение огоньков от центра к краям и обратное схождение
4. Мигание всеми светодиодами

## Алгоритм
- Кнопка обрабатывается с антидребезгом (задержка 200 мс)
- Переменная `n` хранит номер режима (0-4)
- При нажатии `n++`, если `n>4`, обнуляется
- В бесконечном цикле `switch(n)` выполняет соответствующее действие

## Выводы МК
- PD0-PD7 — светодиоды (выходы)
- PB0 — кнопка (вход с подтяжкой)

## Среда разработки
Microchip Studio, частота МК 1 МГц
