# Управление светодиодами с помощью кнопок (Atmega32)

## Задача
Система управления тремя светодиодами с помощью трёх кнопок.

## Алгоритм
Светодиоды мигают (200 мс вкл, 200 мс выкл) при условии:
- K2 отжата
- K1 нажата
- K3 нажата

## Выводы МК
PC2 - выход - Зелёный светодиод
PD4 - выход - Синий светодиод
PB6 - выход - Красный светодиод
PB2 - вход - Кнопка K1 (с подтяжкой)
PA4 - вход - Кнопка K2 (с подтяжкой)
PC4 - вход - Кнопка K3 (с подтяжкой)

## Среда
Microchip Studio, частота 1 МГц
