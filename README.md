# Мои проекты по программированию микроконтроллеров

## О себе
Студент 3 курса Московского Политеха, специальность «Электроэнергетика и электротехника». Ищу летнюю практику программиста микроконтроллеров.

## Стек технологий
- Язык: C
- Микроконтроллеры: Atmega32, STM32
- Среды разработки: Microchip Studio, STM32CubeIDE
- Симуляция: Proteus
- Библиотеки: работа с LCD1602

## Содержание 
- `01_Button_LED` — управление светодиодами кнопками
- `02_ShiftLeds` — бегущие огоньки (два светодиода)
- `03_ButtonModeSwitch` — переключение 5 режимов свечения одной кнопкой
- `04_7segCounter` — 7-сегментный счётчик с длинным нажатием
- `05_Keypad_Safe` — кейпад + LCD + пароль для сейфа
- `06_ADC_LM335` — измерение температуры через АЦП и датчик LM335
