# Управление светодиодами с помощью кнопок (Atmega32)

## Задача
Разработать систему управления тремя светодиодами (зелёный, синий, красный) с помощью трёх кнопок (K1, K2, K3) на микроконтроллере Atmega32.

## Реализованный алгоритм
- Светодиоды загораются все одновременно на 200 мс и гаснут на 200 мс, создавая эффект мигания
- Алгоритм работает только при условии:
  - Кнопка K2 отжата
  - Кнопка K1 нажата
  - Кнопка K3 нажата

## Используемые выводы МК

PC2 - выход - Зелёный светодиод

PD4 - выход - Синий светодиод

PB6 - выход - Красный светодиод

PB2 - вход - Кнопка K1 (с подтяжкой)

PA4 - вход - Кнопка K2 (с подтяжкой)

PC4 - вход - Кнопка K3 (с подтяжкой)

## Ключевые моменты кода

- Настройка портов на ввод/вывод через DDRx
- Включение подтягивающих резисторов через PORTx
- Проверка нажатия через чтение PINx
- Использование _delay_ms() для создания мигания

## Среда разработки

- Microchip Studio
- Симуляция в Proteus
- Частота МК: 1 МГц

## Как запустить

1. Открыть проект в Microchip Studio
2. Скомпилировать (F7)
3. Загрузить прошивку в симулятор Proteus или на реальное устройство
