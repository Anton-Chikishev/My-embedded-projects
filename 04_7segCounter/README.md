# 7-сегментный счётчик с ускорением (Atmega32)

## Задача
При коротком нажатии на PA2 число увеличивается на 1. При длительном нажатии (более 2 секунд) число начинает уменьшаться. PA3 пока не используется.

## Алгоритм
- Кнопка на PA2 с подтяжкой
- При нажатии — задержка 50 мс (антидребезг)
- Запускается счётчик, увеличивающийся каждые 50 мс
- Если в момент отпускания счётчик < 40 — короткое нажатие, `n++`
- Если счётчик >= 40 — длинное нажатие, `n--`
- Значение `n` выводится на 7-сегментный индикатор (общий анод, порт D)

## Выводы МК
- PD0-PD7 — 7-сегментный индикатор (выход)
- PA2, PA3 — кнопки (вход с подтяжкой)

## Особенность
Изменение цифры происходит по отпусканию кнопки, а не по нажатию.

## Среда разработки
Microchip Studio, частота МК 1 МГц
