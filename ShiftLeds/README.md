# Бегущие огоньки (Atmega32)

## Задача
Два светодиода сдвигаются сверху вниз по порту D, затем обратно — снизу вверх.

## Алгоритм
- Битовая маска `(3 << i)` зажигает два соседних светодиода
- Цикл от 0 до 6 (вниз), затем от 6 до 0 (вверх)
- Задержка 250 мс между шагами

## Выводы МК
Все выводы порта D (PD0-PD7) — выходы, подключены к светодиодам

## Среда разработки
Microchip Studio, частота МК 1 МГц
