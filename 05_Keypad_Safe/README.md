# Пароль для сейфа (Atmega32)

## Задача
Ввод 3-значного пароля (666) с кейпада, маскировка звёздочками, индикация верности/неверности на LCD и светодиоде.

## Алгоритм
1. Инициализация портов: столбцы кейпада на выход, строки на вход с подтяжкой
2. Опрос кейпада: по очереди подаём 0 на столбец, проверяем строки
3. При нажатии на дисплей выводится `*`, накопление в переменной `protect`
4. После трёх введённых символов пароль сравнивается с 666
5. При совпадении — моргание светодиодом и вывод "TRUE!", при несовпадении — "FALSE!"
6. Через секунду дисплей очищается, запрос пароля повторяется

## Выводы МК
- Кейпад: столбцы PD0-PD2 (выход), строки PD3-PD6 (вход с подтяжкой)
- Светодиод PC0 (выход)
- LCD на стандартных выводах (из библиотеки lcd1602wire4.h)

## Особенности
- Код возврата кнопки `n=10` означает «нет нажатия»
- Звёздочка и решётка имеют коды 0x2A и 0x23
- Курсор на LCD отображается, символы выводятся по одному

## Среда разработки
Microchip Studio, библиотека lcd1602wire4.h, частота МК 1 МГц
